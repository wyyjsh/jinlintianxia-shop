## 金林天下 商城系统

## 项目参考了<a href="https://github.com/lilishop/lilishop">lilishop商城</a>，仅作学习使用
### 商城介绍
**官网**：https://www.jinlintianxia.cn

金林天下商城系统支持商家入驻，后端基于SpringBoot 研发，前端使用 Vue、uniapp开发， **系统全端全部代码开源**

前后端分离，支持分布式部署，支持Docker，各个API独立，并且有独立的消费者。
